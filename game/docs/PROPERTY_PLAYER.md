# Property (player)

1. Buy a deed at the Real Estate Office (web or in-game).
2. Your deed is an item; the parcel record lives in the sovereign archive.
3. Start an operation (rent / floor / line) to earn periodic credits — on the deed detail page (web) use **Start income**, or in-game `startproperty` (with optional `rent` / `floor` / `line`).
4. Build structures to use slots and unlock upgrades (developer commands / future UX).
5. Visit your parcel: use `openproperty` once on a carried deed to create the interior shell, then `visitproperty` to travel there.

Reference prices on the home dashboard are sovereign list-price analytics, not guaranteed resale offers.
